# Piano-py

Portfolio Piano using pygame + numpy + Python 3.9.

![App screenshot](screenshots/main.png)

If the image does not render on GitHub, use the raw image URL:
https://raw.githubusercontent.com/kacper5drub-oss/Piano-py/4e1beaf9acc5e91902cf32ccf48350767ff6c533/screenshots/main.png

## Features
- Play piano notes using keyboard or mouse
- Synthesized audio with numpy
- Simple GUI built with pygame

## Requirements
- Python 3.9
- pygame
- numpy

Install requirements:

```bash
python -m pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Contributing
Contributions and bug reports are welcome. Open an issue or submit a PR.

If you'd like to submit a small change (recommended workflow):
1. Create a branch:
   ```bash
   git checkout -b docs/add-screenshot-readme
   ```
2. Save this README.md into the repo root (overwriting the existing README.md).
3. Stage, commit, and push:
   ```bash
   git add README.md
   git commit -m "docs: update README and add screenshot"
   git push origin docs/add-screenshot-readme
   ```
4. Open a pull request on GitHub from `docs/add-screenshot-readme` into `main`.

Alternatively, you can edit the README directly in the GitHub web UI: open README.md, click the pencil icon, paste the content above, and commit.
